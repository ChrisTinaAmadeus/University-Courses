# 中文分词系统使用手册

### 一、启动系统
程序启动后，会自动加载字典文件。加载完成后，将显示主菜单：
```
╔══════════════════════════════════════════════════════════╗
║               中文分词系统 (N-最短路径)                    ║
╠══════════════════════════════════════════════════════════╣
║                                                          ║
║    [1] 开始分词                                           ║
║    [2] 退出系统                                           ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝

请输入选项编号 > 
```

### 二、进行分词
1.  在主菜单输入 `1` 并回车。
2.  **输入句子**：根据提示输入需要分词的中文句子（支持长句）。
    *   提示符：`请输入需要分词的句子 > `
    *   示例：`这的确是一个好主意`
3.  **输入 N 值**：输入希望获取的最短路径数量 N。
    *   提示符：`请输入N-最短路径中的N值 > `
    *   输入 `1`：仅输出最优分词结果。
    *   输入 `3`：输出前 3 种可能的分词结果，按概率从高到低排序。
4.  **查看结果**：系统将输出分词结果，词与词之间用 `/` 分隔。
    *   示例输出：
        ```
        ------------------ 分词结果 ------------------
        1 : 这 / 的确 / 是 / 一个 / 好 / 主意 / 
        ----------------------------------------------
        ```
5.  按回车键返回主菜单。

### 三、 退出系统
在主菜单输入 `2` 并回车，即可退出程序。
